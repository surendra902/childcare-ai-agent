Capabilities

What ChildCareAI Can Do

Seven specialised AI tools designed for childcare centre administration

📋

Smart Roster Drafting

Auto-generates draft rosters based on staff availability, room assignments, and qualification requirements. Respects NQS educator-to-child ratios.

🔄

Instant Cover Finder

When staff call in sick, the AI instantly identifies available cover staff ranked by room familiarity and qualification match.

📊

Ratio Compliance Check

Real-time staff-to-child ratio monitoring across all rooms. Flags potential NQS breaches before they happen.

✉️

Parent Communications

Drafts professional, warm parent messages and newsletters with proper tone. All drafts require human review before sending.

⚠️

Director Escalation

Sensitive actions (roster changes, data access) automatically escalate to the director with time-limited approval tokens.

👥

Staff Availability

Query who's available for any date, room, or shift. Filters out staff on leave, already rostered, or without required qualifications.

Built for Childcare Compliance

The agent follows a structured agentic loop: receive message → strip PII → reason with AI → execute tools → check permissions → restore PII → respond. Every action is logged in a tamper-proof audit trail.

  • ✓ PII stripped before AI processing — names never leave the server
  • ✓ Role-Based Access Control (Director / Admin / Educator / ReadOnly)
  • ✓ Time-limited approval tokens for sensitive actions
  • ✓ AES-256-GCM field-level encryption at rest
  • ✓ 7-year audit log retention for regulatory compliance
  • ✓ Rate limiting and security headers (HSTS, CSP, XFO)
